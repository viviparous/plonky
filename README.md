# plonky
A Perl program to generate a conky rc file. 

What is conky? See: https://github.com/brndnmtthws/conky

## Requirements
* Linux
* conky
* Perl

## Usage
In a terminal in $home, execute:
    `perl plonky.pl > .conkyrc`

Then run 
    `conky -d -b`
